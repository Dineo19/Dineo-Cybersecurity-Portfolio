#  Python Port Scanner

A simple yet powerful Python tool that scans a target IP address for open ports.  
This project demonstrates networking knowledge, reconnaissance skills, and Python automation — all essential for cybersecurity and ethical hacking.

---

##  Features

✔ Scans common ports  
✔ Identifies open and closed ports  
✔ Displays clean output  
✔ Uses TCP connection attempts  
✔ Beginner friendly and well-documented
